# CrossTerm Documentation

CrossTerm is a cross-platform terminal emulator and remote access suite for macOS, Windows, and Linux. It consolidates SSH, SFTP, RDP, VNC, serial, Telnet, and local shell management into a single native application.

## Installation

| Platform | Method |
|----------|--------|
| macOS | `brew tap alal76/crossterm && brew install --cask crossterm` |
| Windows | [Download installer (.exe or .msi)](https://github.com/alal76/CrossTerm/releases/latest) |
| Linux | [Download .deb / .rpm / AppImage](https://github.com/alal76/CrossTerm/releases/latest) |
